# Dependencies

[YARN](https://yarnpkg.com/en/docs/install)

# Instructions

```
> git clone https://github.com/fullstackadam/react-user-list.git
> cd react-user-list
> yarn
> yarn run dev-server
```

Go to http://localhost:8080/ in your web browser
